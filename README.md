# COFFEE SHOP SARIS 
            ✨version 2026 - python✨ 

##   _¿Para que sirve?_ 
---
_Este proyecto es un sistema en Python que registra los tipos de cafe, esto permite agregar, editar y eliminar bebidas del menu establecido._

---

## _Funciones Principales_
---

* _**Agregar cafe:** puede agregar cafe nuevo con disponibilidad, descripcion y precio._

* _**Editar cafe:** puede modificar el cafe ya mostrado en el menu, cambiando su disponibilidad, precio, y descripcion en caso de ser necesario._

* _**Eliminar cafe:** se puede eliminar un cafe del menu en caso de que este ya no vaya a estar a la venta._
